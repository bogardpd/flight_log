Overview

Controls User pages and actions.

Since multi-user functionality is not yet implemented in this application, these pages and actions are only used to support a single user and have very little implemented functionality at this time. If multi-user functionality is added, this controller will require substantial modification.

All of these actions currently require a verified user.

def create
  @user = User.new(user_params)
  if @user.save
    flash[:success] = ("User " + ActionController::Base.helpers.content_tag(:strong, @user.name) + " was successfully created!")
    redirect_to root_path
  else
    render "new"
  end
end
